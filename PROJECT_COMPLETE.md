## 🚀 Quick Start

### 1. Local Development (5 minutes)

```bash
# Navigate to project
cd portfolio-site

# Install dependencies (if first time)
npm install

# Start development server
npm run dev

# Visit http://localhost:5173
# Changes update instantly!
```

### 2. Customize Your Content (15 minutes)

Edit these files:
- `src/components/Hero.jsx` - Your name & title
- `src/components/Contact.jsx` - Email & Formspree ID
- `index.html` - Page title
- Other components - Your content

### 3. Deploy (5-10 minutes)

Choose one:

**Option A: Vercel (RECOMMENDED)**
```bash
git init
git add .
git commit -m "Initial setup"
git remote add origin https://github.com/YOUR_USERNAME/portfolio-site.git
git push -u origin main
# Then sign up at vercel.com, connect GitHub repo
# Click deploy - DONE! 🎉
```

**Option B: Netlify**
- Same GitHub setup
- Sign up at netlify.com
- Connect GitHub repo
- Deploy - DONE! 🎉

**Option C: GitHub Pages**
- GitHub setup same
- Build: `npm run build`
- Push dist folder
- Enable Pages in settings
- DONE! 🎉
